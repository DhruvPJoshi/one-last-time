One Last Time

Made with ♥ by Dhruv Joshi

For the moment, let's go back to first year.

Remember that time, when we were introduced to the 'C programming language' for the first time?

Yes. I'm talking about that very first "Hello World" program.

The key point we learned from that program was:

“Program execution begins from main() method.”

I want you to pay a closer attention here, because our life is very much similar to this "main()" method.

We all know that when we try to execute a program,

it is first loaded inside the computer's main memory.

After that, the main() method's execution begins.

In a very similar fashion, when we are born, when we come into existance,

we are loaded into this gigantic memory, this world,

and the main() method, our life, begins its execution.

If you feel convinced, at all, with the forementioned explanation,

then I'm very grateful to have someone who understands what I'm trying to say here.

And for those who didn't feel convinced at all...

"Are bola na relative hai"

Anyway, let's move ahead. Shall we?

Um. Functions. Let's talk a little bit more about them.

Almost everything we do in our life, every task we perform, are like function calls.

There are some tasks, like predefined functions, that we perform everyday,

without having any idea about how that happens. All we know is that it just happens.

Like Dreams. We know we see them. But we don't know "How?" or "Why?"

And there are also some tasks, that we perform everyday,

with some level of knowledge on how "something" happens.

There are many such tasks, like WakeUp(), Move(), PickUp(), Eat(), Run(), Play(),... And the list goes on.

One of such function calls is,


                    function College() {
                        // Our 4 years
                        // are right here
                        ...
                        ...
                    }

Which we are about to finish executing.

"Wait. What's all this? Did we even call this function?"

"My answer to your latter question would be, yes, about 4 years ago, when we entered our college and became part of this campus, till now. And for your former question, you'll have to read further."

"Do you remember our first day of college?"

"Ah. The student orientation program. 28th July, 2014. How can we forget that."

"Woah. You even remember the date?! That's surprising. But yeah. That's the day. The day when we all met each other for the first time. Hopefully. We had also introduced ourselves during that orientation program. It was as if we were declaring ourselves as variables, which were going to exist inside this college campus, for the next few years. It sure is an amazing memory."

"Yeah. Going on the stage and introducing yourself to everyone for the first time."

"You bet. Not everyone has the courage to do that on their very first time. But we were able to manage that somehow."

"Yeah. We have gone through things more difficult than that. Remember the devils that haunts every student. ('We don't speak his name, Harry.')."

"Are you by any chance talking about Assignments, Viva, and Exams?"

"Of course I am. And even that wasn't enough, they introduced 'Design Engineering' and 'Final Year Project'. And how lucky we are. Our college had to introduce 'Mini Project' too."

"But there were good times too."

"Mhmm. Navratri, Sports festival, Day's celebrations, and Birthday celebrations too."

"Yes. And don't forget to add those failed plans. And the times when we used to group study. And the times when we had to work our arses off, for project submissions. And those last minute preparations before exams. And always asking this question: 'Yaar, viva he. Practical to nahi karvayenge na?'"

So many memories.

Now we've reached that final stage.

A final line that remains to be executed.

The end of this wonderful journey.

But wait.

There is something written over there, just before that return statement.

A comment.


                    function College() {
                        ...
                        ...
                        // Store and cherish every single memory
                        // you get in return from here.
                        // They are precious after all.
                        return _____;
                    }

I remember a quote, from my favourite fantasy novel, "The Lord of The Rings: The Return of The King", that says:

“How do you pickup the threads of an old life? How do you go on... ...when in your heart you begin to understand... ...there is no going back? There are some things that time cannot mend... ...some hurts that go too deep... ...that have taken hold.”

If you've seen our class group's DP, then you've read this already:

"Engineering is not a destination. It's a beautiful journey!"

It sure is.

“An array isn't an array, if there are no elements in it. Similarly, this journey wouldn't have been this wonderful, without you guys in it.”

"So. This is it."

"No. Not quite. There's room for a little more."

Our journey, until now, has ended.

But a new journey awaits.

And there are many more journey's to come.

But I'm really going to miss this one.
